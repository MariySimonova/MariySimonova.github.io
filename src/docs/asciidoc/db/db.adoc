= Структура данных сервиса работы с задачей

Структура данных, используемая сервисом работы с задачами, приведена на диаграмме.

[plantuml, db, png]
----
include::db.puml[]
----

.Задача
|===
|*Атрибут* |*Ключ* |*Тип* |*Не пустой* |*Описание*

|Заголовок
|title
|Однострочный текст
|true
|Краткое описание задачи.

Длина на более 255 символов

|Описание
|description
|Многострочный текст
|false
|Полное описание задачи, что необходимо сделать.
|===

.Статус
|===
|*Атрибут* |*Ключ* |*Тип* |*Не пустой* |*Описание*

|Статус
|status
|Однострочный текст
|true
a|Статус исполнения задачи.

Одно из значений:

* `todo` - к исполнению;
* `in progress` - в работе;
* `done` - исполнена;
* `declined` - отменена.

По умолчанию `todo`
|===
